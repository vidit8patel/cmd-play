Flappy Bird Game

To play Flappy Bird, you need to run it on a local server.

Follow these steps to run Flappy Bird:

npm install -g http-server cd /Users/vidit/Desktop/hackathon/Flappy-Bird http-server -p 8080

Then open http://localhost:8080 in your browser.

Once the server is running, you can play the game directly in your browser!
